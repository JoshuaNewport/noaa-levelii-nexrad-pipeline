# Code of Conduct

We want this project to be a friendly, professional place to collaborate on NEXRAD Level II processing.

## Expectations

* Be respectful and constructive.
* Focus on the code and the science, not the person.
* Accept feedback in good faith and give it professionally.
* No harassment, discrimination, or personal attacks.

## Enforcement

Project maintainers may remove content or restrict participation that violates these guidelines.
Serious or repeated violations may result in a temporary or permanent ban.

## Reporting

If there is a serious issue, contact the project maintainer at:
**joshua@weather-nomad.com**

Reports will be handled as fairly and privately as possible.
